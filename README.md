# SINATRA API WITH JWT AUTH

## Installation
```
bundle install
```

## Start without docker
```
bundle exec rackup
```

## Tests
```
bundle exec rspec
```

## Bundle audit
```
bundle-audit update
bundle-audit check
```

## Docker
```
API_PORT=2900 docker-compose up
```
